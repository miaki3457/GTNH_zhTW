# GTNH.js 使用指南
1.進入[JSZip](https://stuk.github.io/jszip/)的首頁<br />
2.F12打開控制台<br />
3.將GTNH.js的內容粘貼進控制台內<br />
4.輸入`parse(true, true)`獲得`en_US.lang`和覆蓋用文件，或輸入`parse(true, false)`獲得`result.lang`(白版用於漢化的文件)<br />
